La jardinería es una actividad que ofrece diversión, así como ayuda a tener buena salud. Es una actividad física que puede enriquecer la vida de una persona de edad avanzada tanto espiritual como mentalmente. Así, la jardinería puede ser una gran manera de pasar un poco de tiempo y obtener algo de bronceado.
